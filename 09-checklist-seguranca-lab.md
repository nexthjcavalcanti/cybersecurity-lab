# 09 - Checklist de segurança do laboratório

## Antes de publicar no GitHub

Verificar se o conteúdo não mostra:

- IP público
- IPv6 público completo
- IP privado completo
- endereço MAC
- nome da rede Wi-Fi
- nome do computador
- usuário do Windows
- e-mail pessoal
- tokens
- senhas
- chaves de API
- prints completos do terminal
- dados de clientes ou empresas

## Como mascarar dados

Exemplos:

- IP privado: `192.168.15.x`
- IP externo: `IP_externo_mascarado`
- IPv6: `IPv6_mascarado`
- Usuário: `usuario_local`
- E-mail: `email_mascarado`

## O que aprendi

Em cybersecurity, documentar bem é importante.

Mas documentar com segurança é ainda mais importante.

Antes de publicar qualquer estudo, preciso revisar se estou expondo alguma informação sensível.
