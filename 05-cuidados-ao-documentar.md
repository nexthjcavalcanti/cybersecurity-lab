# 05 - Cuidados ao documentar estudos de Cybersecurity

## Objetivo

Criar o hábito de documentar estudos sem expor informações sensíveis.

## Informações que não devo publicar

* IP público
* IP completo da minha máquina
* endereço MAC
* nome da rede Wi-Fi
* dados do roteador
* e-mails pessoais
* senhas
* tokens
* chaves de API
* prints completos do terminal
* dados de clientes ou empresas

## Boas práticas

* Mascarar dados sensíveis
* Usar exemplos fictícios
* Revisar prints antes de postar
* Não publicar arquivos com credenciais
* Separar ambiente de estudo de ambiente pessoal

## O que aprendi

Em cybersecurity, não basta aprender ferramentas.

Também é necessário desenvolver cuidado com exposição de dados, privacidade e segurança operacional.
