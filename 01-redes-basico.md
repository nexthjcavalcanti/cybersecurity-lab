## Conceitos básicos

**IP:** é o endereço de um dispositivo na rede.

**DNS:** traduz nomes de sites, como google.com, para endereços IP.

**HTTP:** protocolo usado para acessar sites.

**HTTPS:** versão segura do HTTP, com criptografia.

**Porta:** número usado para identificar um serviço em um computador.

**TCP:** protocolo mais confiável, usado quando a entrega correta dos dados é importante.

**UDP:** protocolo mais rápido, usado quando velocidade é mais importante que confirmação de entrega.

## Portas comuns

* Porta 80: HTTP
* Porta 443: HTTPS
* Porta 22: SSH
* Porta 53: DNS
* Porta 25: SMTP
* Porta 3306: MySQL

## Relação com Cybersecurity

Entender redes é essencial para identificar serviços expostos, analisar conexões, interpretar alertas e encontrar possíveis vulnerabilidades.
