# 03 - Comandos de rede

## Comandos testados

### ping

Comando usado:

`ping google.com`

Resultado:

* Pacotes enviados: 4
* Pacotes recebidos: 4
* Perda: 0%
* Tempo médio: 43ms

O que aprendi:

O comando `ping` testa se meu computador consegue se comunicar com outro servidor na rede ou na internet.

### nslookup

Comando usado:

`nslookup google.com`

Resultado:

* IPv6 encontrado: 2800:3f0:4001:837::200e
* IPv4 encontrado: 172.217.29.174

O que aprendi:

O comando `nslookup` consulta o DNS para descobrir o endereço IP de um domínio.

## Resumo

O `ping` testa conectividade.

O `nslookup` consulta informações de DNS.

Esses comandos são importantes para redes, troubleshooting e cybersecurity.
